Extrair essências e cultivar ontologias de software
-----------------------------------------------------

Estou aprofundando meus conhecimentos sobre migração de dados e, atualmente, estudo engenharia e arquitetura de software agnósticas para diversos padrões de projetos, com o objetivo de me tornar um desenvolvedor versátil que entenda os programas como ferramentas de movimentação de dados.

* 🚀 Atualmente, estou trabalhando em [análise exploratória e migração de dados no Departamento de Migração de Dados da Virtual Informática](http://VirtualMigration )

* 👯 Tenho me concentrado no desenvolvimento de ferramentas ETL internas para o departamento.

* 🧠 Atualmente, estou ampliando meus conhecimentos em engenharia de dados, ciência de dados e bancos de dados relacionais.

* 👥 No momento, estou me dedicando ao estudo de DevOps para automatizar processos de migração; com esse conhecimento, posso ajudar a automatizar tarefas em qualquer setor. Envie-me uma mensagem se quiser colaborar.

* 💬 Vamos conversar sobre os últimos avanços em tecnologia, complexidade algorítmica, softwares baseados em Unix e Linux, bem como ferramentas de código aberto e auto-hospedadas.

* 🧠 Leio filosofia como hobby, de Platão a Nietzsche, de Schopps a Kant, de Marx, Durkheim e Weber a Sabrina, Florestan e Jessé. Sem falar em Turim, Church, von Neumann, Searle, Dreyfus, Chomsky e assim por diante.

* ✉️ Você pode entrar em contato comigo pelo e-mail [dev\_helbert@virtualloja.com.br](mailto:dev_helbert@virtualloja.com.br)
